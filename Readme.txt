此工具名為:Name．Tool．Random，簡稱為NTR。

由於檔案為jar檔案，需要有jre才能啟動。

用途為將特定資料夾中的所有檔案重新命名。

命名規則為:從0開始，會一直遞增，不會更動到附檔名，方式為隨機，每次的結果都不同。

資料夾內最好不要有子資料夾，否則會將子資料夾一起重新命名。

命名無法復原，請慎重使用。

使用方式為開啟NTR.jar後，輸入目的地資料夾，並且按下Enter。

請輸入完整路徑，例如:C:\Users\user\Desktop\123。

建議先複製好路徑後，在視窗內使用ctrl+v貼上。

如果路徑正確，按下Enter後視窗就會消失，並且將該資料夾中的檔案重新命名。
